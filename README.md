# AltarChal

## Core Dependencies
* Angular/cli
* NodeJs 12+

## Install
* Open root of project in terminal and then:
```
> cd BE
> npm install
> cd ../FE
> npm install
```

## How to Run

* Open root of project in terminal 1 and then:
```
> cd BE
> npm start
```

* Open root of project in terminal 2 and then:
```
> cd FE
> npm start
```
